## Part F: Linux Research

Linux is an open-source operating system widely used in servers and cybersecurity.

Linux is important in cybersecurity because it provides powerful terminal tools and is used in penetration testing tools like Kali Linux.

Linux is different from Windows because it is open-source, more secure, and heavily terminal-based.

A Linux distribution is a version of Linux like Ubuntu, Kali Linux, or Parrot OS.

Ethical hackers prefer Linux because it is powerful, flexible, and supports security tools.
