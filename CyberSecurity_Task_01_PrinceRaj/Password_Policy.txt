PASSWORD SECURITY POLICY

1. Use passwords with a minimum length of 12 characters.
2. Include uppercase and lowercase letters.
3. Include numbers and special characters.
4. Avoid using personal information such as names, birthdays, or phone numbers.
5. Use a unique password for every account.
6. Enable Multi-Factor Authentication (MFA) whenever available.
7. Never share passwords with anyone.
8. Change passwords immediately if a compromise is suspected.
9. Use a trusted password manager to store passwords securely.
10. Avoid storing passwords in plain text files.

Answers:

1. What makes a strong password?
   A strong password is long, unique, and contains a combination of uppercase letters, lowercase letters, numbers, and special characters.

2. Why should passwords be unique?
   Unique passwords prevent attackers from gaining access to multiple accounts if one password is compromised.

3. What is Multi-Factor Authentication (MFA)?
   MFA is a security method that requires users to provide two or more verification factors before gaining access to an account.

4. What are the risks of password reuse?
   If one account is compromised, attackers can use the same password to access other accounts.
